## [1.0.0] - 2024-12-01

### Added
- **Core Features**
  - Plant milk rating system with 1-5 star ratings
  - Product database with brands, names, and flavors
  - Barista edition product support
  - Photo capture for milk tests
  - Notes and shop location tracking
  - Price/quality ratio ratings

- **Social Features**
  - Community feed with user submissions
  - Like and comment functionality
  - User profiles with avatars
  - Notification system for interactions

- **Discovery**
  - Product search with fuzzy matching
  - Filter by brand, flavor, and properties
  - Sort by rating, date, and name
  - Aggregated results view
  - Interactive world map showing test locations

- **User Experience**
  - Progressive Web App (PWA) support
  - Mobile-responsive design
  - Dark/light theme support
  - Country-based filtering
  - Barcode scanner for products

- **Authentication**
  - Email/password authentication
  - Password reset functionality
  - User profile management
  - Notification preferences

### Technical
- Built with React 18, TypeScript, and Vite
- Supabase backend with PostgreSQL
- Row Level Security (RLS) policies
- Tailwind CSS with custom design system
- Framer Motion animations
- Mapbox integration for world map
